Tutorials

by Alex Batovsky

Posted on February 10, 2015 at 9:00 PM
This is an exmaple of what your angular application could look like!

Code Snippets:

(function() {
var app = angular.module('myApp', []);

app.controller('TutsController', function(){
this.tutorials = tuts;
});

var tuts = [
{
"url": "http://tutorials.jenkov.com/angularjs/index.html",
"title": "AngularJS Tutorial",
"image": "http://tutorials.jenkov.com/images/layout/small-portrait_mini.jpg",
"author": "Jakob Jenkov"
},
...
This is creates a variable named app and sets it to an anuglar model named myApp. We can then use the app variable to create a controller named TutsController that can be used in html. We can then use the function of the controll to set the current state of tutorials in the controller to an array of data that contains information about angular tutorials.
This creates an unordered list that has an ng-repeat telling the broswer to display every tutorial that is in your array of data. This works because your HTML is using the controller you created from the last step and cycling through the array of data. You can use 'filter:' and 'orderBy:' which are built in angular functions that allow you to search, filter and sort your data, as well. Using angular expressions (double curly braces) allow you to call each piece of information from your array of data which allows you to display it to the page. Please note that you must add 'ng-app="myApp"' to your html tag as well as 'ng-controller="TutsController" as (anything you want)' to your body tag.
